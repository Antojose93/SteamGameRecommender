Application Endpoints

  • Home: / - API's Home Page
  • Developer: /developer/{developer} - Developer information
  • User Data: /userdata/{user_id} - User data
  • User for Genre: /userforgenre/{genre} - User with most playtime for a genre
  • Best Developer: /bestdeveloper/{year} - Top 3 best developers for a year
  • Sentiment Analysis: /sentimentanalysis/{year} - Sentiment analysis of reviews for a year
