Application Endpoints List

  • Home: / - API's Home Page
  • PlayTimeGenre: /playtimegenre/{genre} - Year with most played hours for a genre
  • UserForGenre: /userforgenre/{genre} - User with most played hours for a genre
  • UsersRecommend: /usersrecommend/{year} - Top 3 games most recommended by users for a year
  • UsersWorstDeveloper: /usersworstdeveloper/{year} - Top 3 worst developers for a year
  • SentimentAnalysis: /sentimentanalysis/{year} - Sentiment analysis of reviews for a year
  • RecomendacionJuego: /recomendacion_juego/{id} - Recommendation of similar games to a given game
